# Atrato interview practice

Before start run
```sh
npm run prepare
```
This command install all dependencies and build the project.

## Start the server
```sh
npm run start
```
To change the port use the enviroment variable `PORT`, like
```sh
PORT=80 npm run start
```
## Other commands
### Storybook server
```sh
npm run storybook
```
### Testing
```
npm run test
```
Get coverage report
```sh
npm run coverage
```
