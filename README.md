# Atrato interview practice

Before start install dependencies.

```sh
npm install
```

And build the project.

```sh
npm run build
```

## Start the server

```sh
npm run start
```

To change the port use the enviroment variable `PORT`, like

```sh
PORT=80 npm run start
```

## Other commands

### Storybook server

```sh
npm run storybook
```

### Testing

```
npm run test
```

Get coverage report

```sh
npm run coverage
```
